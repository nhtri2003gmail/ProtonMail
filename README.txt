Requirement: Python 3.x
Link: https://www.python.org/downloads/

This program help you create a ProtonMail account for ProtonVPN
Just run it, the credential for ProtonVPN will be saved in ProtonMail.txt
After successfully creating a Proton mail, log in to ProtonVPN with that account and enjoy!!!
